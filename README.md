# Maruwá Villas - Landing Page Premium 🌴

Bem-vindo(a) ao repositório oficial da Landing Page **Maruwá Villas**. Este projeto foi desenvolvido com foco em alta performance, design sofisticado (Luxo/Boutique) e conversão, proporcionando uma experiência imersiva e de alto padrão para os usuários.

## 🚀 Visão Geral

A página foi construída para refletir a exclusividade das hospedagens da Maruwá no Litoral Norte da Bahia. Ela conta com animações fluidas baseadas em rolagem (scroll), carrosséis dinâmicos e formulários integrados para captura de leads e reservas.

## ✨ Principais Funcionalidades

* **Design Glassmorphism:** Elementos translúcidos e modernos que remetem à sofisticação.
* **Animações Avançadas:** Integração com GSAP e ScrollTrigger para revelação de elementos à medida que o usuário desce a página.
* **Carrosséis Interativos:** Utilização do Swiper.js para portfólio de imóveis, leitura fluída de depoimentos (estilo *marquee*) e exibição de selos de confiança.
* **Modais de Galeria:** Visualização ampliada de imagens e detalhes das propriedades sem sair da página atual.
* **Formulário de Reserva Inteligente:** Integração com Flatpickr para seleção intuitiva de datas de estadia.
* **100% Responsivo:** Experiência perfeita em smartphones, tablets e desktops.

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando as melhores e mais modernas tecnologias web para garantir velocidade e facilidade de manutenção:

* **HTML5** (Semântico e estruturado)
* **Tailwind CSS** (via CDN para estilização rápida e responsiva)
* **JavaScript Vanilla** (Lógica de modais, menus e interações)
* **GSAP & ScrollTrigger** (Animações de alta performance)
* **Swiper.js** (Carrosséis de imagens e cards)
* **Flatpickr** (Calendário para formulário de reservas)

## 💻 Como Visualizar e Executar o Projeto

Como o projeto utiliza tecnologias Front-End estáticas (HTML/CSS/JS), ele não requer a instalação de servidores complexos ou bancos de dados locais.

1. Faça o download ou clone este repositório.
2. Descompacte os arquivos (se estiver em `.zip`).
3. Dê um duplo clique no arquivo `index.html` para abri-lo diretamente no seu navegador padrão (Google Chrome, Safari, Edge, etc.).

## 🎨 Guia Rápido de Personalização (Para o Cliente)

Se você precisar alterar textos ou imagens no futuro, o código foi organizado de forma semântica.

### Alterando Textos

Abra o arquivo `index.html` em um editor de código ou bloco de notas estruturado (recomenda-se o *VS Code*). Pressione `Ctrl + F` (ou `Cmd + F` no Mac) e digite o texto que deseja alterar. Substitua o conteúdo mantendo as tags ao redor intactas.

### Alterando Imagens e Vídeos

As mídias do site estão localizadas na pasta `assets/`. Para trocar uma imagem:

1. Adicione a nova imagem na pasta correspondente dentro de `assets/`.
2. Recomenda-se o uso do formato `.webp` para garantir um carregamento rápido.
3. No arquivo `index.html`, localize o caminho da imagem antiga (ex: `src="assets/villa mare/vila-mare01.webp"`) e altere para o nome do seu novo arquivo.
